## Description

This is a quite simple API, built to take part in NTT Data's tech challenge, in which we consume omdbapi's API to retrieve and sanitize data from movies, while sending back a DTO to our client.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```
